# 수정/변경 내역

## 2024-10-21
- Gradle wrapper 스크립트에서 `GRADLE_USER_HOME` 기본 경로를 프로젝트 루트의 `.gradle` 폴더로 강제 지정해 샌드박스 환경에서도 `./gradlew bootRun`이 사용자 홈 디렉터리를 쓰지 않고 실행되도록 수정함.
- `application-local.yml`의 `access.key`, `jwt.secret`를 암호화 문자열 대신 환경변수 우선 값(기본값 포함)으로 전환해 로컬 실행 시 `JASYPT_ENCRYPTOR_PASSWORD` 미설정으로 인한 복호화 오류를 방지함.
- `JwtProvider`가 Base64 디코딩에 실패하면 SHA-512 해시를 사용해 키를 도출하도록 개선하고, `application-local.yml` 기본 JWT 시크릿을 Base64 문자열로 교체해 HS512 키 생성 시 예외가 나지 않도록 함.
